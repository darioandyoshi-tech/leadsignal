### 🚀 **Core Capabilities Available:**

#### **Inbox Management:**
- Create, list, get, update, delete inboxes
- Scalable, API-first email accounts for AI agents
- Multiple inbox support for different purposes/projects

#### **Message Operations:**
- Send, receive, and manage emails as Message objects
- Full SMTP/IMAP compatibility through API
- Attachment handling (send and receive)
- Message tracking and threading

#### **Advanced Features:**
- **Drafts:** Create, manage, and send drafts (human-in-the-loop workflows)
- **Labels:** Powerful labeling system for workflow management
- **Lists:** Allow/block lists for sender/recipient control
- **Webhooks:** Real-time event notifications
- **WebSockets:** Instant email event streaming
- **Pods:** Multi-tenant email management
- **Permissions:** Fine-grained API key permissions
