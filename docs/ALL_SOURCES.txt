This document lists every data source currently wired into the LeadSignal
scraper pipeline. Each row maps to a module under scraper/sources/.

STATUS KEY
----------
LIVE      = Module fetches/ parses live public data on every run.
PLACEHOLDER = Source documented but requires paid account, API key, or browser automation.
STUB      = Uses seed/ fallback data until a live feed is available.
BLOCKED   = Public site blocks scraping; module uses an approved bypass/ alternative.
