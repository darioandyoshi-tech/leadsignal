LeadSignal Permit Coverage
==========================

Active Sources
--------------
- permitstack_omaha        : Commercial live permit API for Omaha metro.
- civicdata_inspections    : ~883K inspection history records (FAIL/PASS events with contractor/address).
- civicdata_permits        : Historical 2006-2007 Omaha permit backfill.
- accela_omaha             : Seed-data stub; real Accela portal needs browser automation.
- odr_feeds                : RSS feeds for building/electrical/mechanical/plumbing/wreck permits + public notices.
- sarpy_smartgov           : Placeholder for Sarpy County Citizenserve portal.

Permit Types by Signal
----------------------
SignalType.permit_filing includes:
  - New building / construction permits
  - Electrical, mechanical, plumbing, wrecking permits
  - Inspection events (passed/failed/re-inspection)
  - Additions, remodels, tenant improvements

Permit Trade Coverage
---------------------
- Electrical
- Mechanical / HVAC
- Plumbing
- Wrecking / demolition
- General building / construction
- Roofing / siding (when listed in description)
- Landscaping / excavation (when listed in description)
- Commercial tenant improvements

Data Gaps
---------
- Live current City of Omaha permits are only available through Accela JS portal or PermitStack (limited sample).
- Sarpy County live permits require Citizenserve login/session automation.
- Historical CivicData current datastores (2016–present) are empty; rely on inspection history for recent activity.
